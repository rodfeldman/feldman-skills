# Feldman Skills

Repositório público de skills e squads compartilhados por **Rodrigo Feldman** ([@rodfeldman](https://github.com/rodfeldman)).

Cada pasta aqui é uma skill ou squad pronta pra usar com Claude Code, AIOX ou ferramentas compatíveis.

## Skills disponíveis

### 📊 [presenter](./presenter)
Skill para gerar apresentações HTML profissionais a partir de qualquer conteúdo. Inclui template, guia visual e prompt gerador.

**Como instalar:** ver [presenter/INSTALL.md](./presenter/INSTALL.md)

---

## Como usar uma skill

1. Baixe a pasta da skill que quer (clique no nome dela acima)
2. Siga o `INSTALL.md` dentro da pasta
3. Pronto, é só usar

## Como baixar uma pasta específica

Se você não quer clonar o repositório inteiro, use [DownGit](https://downgit.github.io/) — cole o link da pasta da skill e baixe só ela como `.zip`.
