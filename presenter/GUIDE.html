Claude Code Skill

Presenter Skill

Dual-screen synchronized presentation pra Zoom e Meet com teleprompter pessoal.

O que faz

Duas telas sincronizadas

Gera duas páginas HTML a partir dos seus slides + roteiro. Uma limpa pra plateia, outra rica pra você. Sincronizam via localStorage — sem servidor, offline, mesmo navegador.

Janela da plateia

slides-{nome}.html

Slides limpos, sem ruído. É o que você compartilha na tela do Zoom ou Meet. Recebe um patch de sync que escuta a outra janela.

Pra compartilharSem distrações

Janela do apresentador

apresentador-{nome}.html

Sua vista privada: slide atual + próximo (miniaturas via iframe), roteiro completo de falas, relógio, cronômetro e navegação por teclado.

Só pra vocêTeleprompter
Quando usar

Apresentação pronta + teleprompter

  • Você já tem slides HTML + um roteiro em Markdown com suas falas, e quer apresentar com teleprompter pessoal sem revelar nada pra plateia.
  • Pedidos do tipo "presenter view", "duas telas slides + roteiro", "compartilhar slides no Zoom e ver minhas falas".
  • Quer rodar tudo offline, sem servidor, sem app extra — só dois arquivos HTML no seu navegador.
Instalação

Um comando e pronto

Descompacte o zip, mova a pasta pra dentro de ~/.claude/skills/ e reabra o Claude Code.

macOS · Linux # dentro da pasta onde você descompactou mv presenter ~/.claude/skills/
Windows · PowerShell Move-Item presenter $env:USERPROFILE\.claude\skills\
No Claude Code /presenter
Como usar

Três passos simples

01

Prepare slides + roteiro

Tenha um arquivo HTML com seus slides e um Markdown com as falas — uma seção por slide.

02

Invoque /presenter no Claude Code

A skill lê os dois arquivos e gera slides-{nome}.html e apresentador-{nome}.html sincronizados.

03

Abra os dois em janelas separadas

Mesmo navegador. Compartilhe a janela de slides no Zoom, mantenha a do apresentador na sua tela. Passar um slide sincroniza a outra automaticamente.
